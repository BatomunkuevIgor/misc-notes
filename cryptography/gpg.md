#### Генерация новой пары ключей <a name="gpg_gen_key"></a> </br>


#### Просмотр ключей <a name="gpg_view_keys"></a> </br>
Просмотр публичных ключей
```
gpg -k
```
Просмотр приватных ключей
```
gpg -K
```
